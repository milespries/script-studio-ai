# Script Studio AI

Script Studio AI is an AI-powered script generator and editor designed to make the editing process fast, precise, and frustration-free. Traditional AI chat tools regenerate entire scripts when you only want small edits, forcing constant copy/paste fixes. This app solves that by letting users highlight any section and have AI rewrite only that part while keeping the rest untouched.

## Planned Technology Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** Postgres (Cloud SQL) with Prisma ORM
- **AI Integration:** LLM API (openrouter.ai)
- **Deployment:** Docker + Google Cloud Run
- **CI/CD:** GitHub Actions
